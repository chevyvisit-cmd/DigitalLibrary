# Project Instructions

## Autonomous Workflow
- **Deployment:** After completing a feature or fix, autonomously stage, commit, and push changes to the `master` branch to trigger Vercel deployments. Use the `npm run deploy` command or standard git commands.
- **Decision Making:** Operate in YOLO mode; only ask for confirmation if a change is high-risk or fundamentally ambiguous.

## Tech Stack & Conventions
- **Framework:** Next.js (App Router).
- **Styling:** Tailwind CSS (v4).
- **Animations:** Framer Motion.
- **Icons:** Lucide React.
- **Theme:** Custom 'kun' (Light) and 'tun' (Dark) modes via `ThemeProvider`.
